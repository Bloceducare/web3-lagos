# Web3 Lagos Conference Repository

## How to Run the App

### Development

```bash
yarn dev
# or
pnpm dev
```

### Production Build

```bash
yarn build
# or
pnpm build
```

### Run Production Build

```bash
yarn start
# or
pnpm start
```

## How to Run the Docker Image

### Development

```bash
docker-compose up --build --force-recreate
```
